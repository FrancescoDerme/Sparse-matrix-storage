Detailed Description

template<NumericOrComplex T, StorageOrder S>
class algebra::COOmap< T, S >

Represents a matrix in Coordinate Map (COOmap) format.

Template Parameters
TThe type of the matrix elements (numeric or complex).
SThe storage order (row-major or column-major).
